üéØ HOW TO USE THE UI
================================================================

1. GENERATE A POST:
   - Type topic in input field
   - Press Enter or click "Generate Post"
   - Wait for AI to generate content

2. REVIEW THE POST:
   - Read the generated content
   - Check character count (280 max for Twitter)
   - Decide: Approve, Reject, or Regenerate

3. APPROVE & PUBLISH:
   - Click "‚úÖ Approve & Publish"

   - Notification confirms success

4. VIEW PENDING:
   - Scroll to "Pending Posts" section
   - See all posts awaiting approval
   - Click Refresh to update list

5. MANAGE PENDING:
   - Click ‚úÖ Approve on any post

üîß TROUBLESHOOTING
================================================================

PROBLEM: UI shows "Cannot connect to API"
FIX: 
   - Check Railway deployment is running
   - Open /health endpoint: https://your-url.railway.app/health
   - Check browser console for errors (F12)

PROBLEM: Generate button does nothing
FIX:
   - Make sure you entered a topic
   - Check browser console (F12) for errors
   - Verify API is responding: curl your-url.railway.app/health

PROBLEM: Pending posts not showing
FIX:
   - Click the Refresh button
   - Check if you have any pending posts
   - Generate a new post first

PROBLEM: UI looks broken
FIX:
   - Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
   - Clear browser cache
   - Try different browser
